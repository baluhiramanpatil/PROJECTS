In train data set,we have 143 columns and in test data set,we have 142 columns. Core data fields of this dataset 1.ID: a unique identifier for each row. 2.Target : indicates the groups of income levels 1:extreme poverty 2:moderate poverty 3:vulnerable households 4:non-vulnerable households 3.parentesco1 : indicates if this person is the head of the household. 4:idhogar : this is a unique identifier for each household. This can be used to create household-wide features, etc. All rows in a given household will have a matching value for this identifier.
